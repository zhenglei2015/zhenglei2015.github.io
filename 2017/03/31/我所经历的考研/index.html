我所经历的考研

Veröffentlicht am 2017-03-31

2017 考研终于结束了，我也没想到是这样一种结局，一种意想之外的完美，就连没有报名学硕也是以一种意外的结局收场，可是面对未来，还是一无所知。

考研的开始

我躺在寝室的床上，今天是我的毕业典礼，可是已经九点多了，手机振振的响着，可是我一点都不想接电话，我知道那是辅导员在催我。昨天又喝了不少酒，继续装睡吧。我从来没想到我这样就毕业了。四年真的好快。我以为我还刚刚开始我的大学。

我的大学几乎是一场空白。没有恋爱，没有奖学金，甚至，没有朋友。我也不知道我是如何度过这四年的。怎么想都是空空如也。面对社会，我有什么呢？我没有户口，只会写写代码，手里握着一个非常不想去的 offer。

。。。。。。。。。。。。。。

算了，考研吧，我告诉我自己。可是，我工作了。

我的工作

工作工作工作，这是一份充满了挑战和意外的工作。

offer我是打比赛拿来的，大众点评的黑客马拉松，搞了个最佳个人，所以可以挑选点评所有的部门。其实一开始我选了算法小组。

那天，算法的老大来面试我了。

“Java 会写吗？”

“不会”

“Jvm 了解过吗”

“没有”

“可以来实习吗？”

“我要做毕设”

。。。。。。。

所以我来到了架构组。

其实刚开始是在架构组的工具小组里面开发 code 平台，就是类似 github 的一个点评内部的代码管理平台，对接了一些发布功能，做了还没有一个月，先后送走了两个 leader 后，工具组消失了，我也开始加入架构组做些中间件这样比较高级的事情。

刚开始用 zookeeper 做一个配置中心，保管了点评上千个项目的配置，研究 zookeeper 让我见到了一个更大的世界，paxos 啊 yahoo 研究院啊，Stanford 去年的 best paper 啊。以及，在巨大流量超多人员的情况下如何管理项目，分配流量。其实这个项目没啥东西了，都是成熟的，可能就是老大让我去练手吧，学习了一段时间的 Java，然后开始写 Redis 相关的东西。做一个缓存平台。Redis 也是超多的事情，学习起来也是很累的，尤其是我们还要做源码级别的修改，又学了一大堆的 HA 啊，备份啊，同步啊这种东西，感觉分布式系统简单的部分都涉猎过了。一晃，就是一年。

2015年7月份到2016年7月份，我的工作生涯结束了。

开始准备

2016年8月，正式跟老大递交了辞呈，挽留的话也听了一些，但是考研的决心还是没有变得，而且我想，有了工作经验，考研复试的时候会有优势吧（这件事情最后验证为真）。辞职回到学校，买了好多好多的书，查了好多的备考资料，可是现在全忘记了，基本就是那些每年迎来一批又一批学生的考研名师们和那些年年爆款的书，可是和那些已经复习了半年甚至多年的人相比，我一点都不懂什么是考研。

其实也没有想太多，如果想太多的话，结论应该是不要考。每天早上起床七点半到晚上十一点半睡觉，中间就不停的看书做题看书做题，比高三都要累。还要担忧时间不够，最后整理东西的时候我发现，用过的书竟然有一米多高，其中有些书还做了不止一遍。关于那段日子的回忆，只有不可思议，压力爆炸，无人理解，甚至有时候一天到晚只和卖饭的阿姨说过话，而所谓的饭，几乎每天都一样，韩式炒鸡块，这道菜我吃了几十次。下午五点半之后去跑步。我真的是不知道那些日子是怎么过来的。累与痛苦，伴随着闹钟每天淹没着我，以至于，最后都麻木了吧。

上考场了，上考场了，紧张不安又有点期待。在考场上每科 3 小时，我用尽了我每一个脑细胞，以至于那几天我根本吃不下东西，只能喝粥。

终于，我考完了，放空了。

等成绩

等成绩的日子，比考试之前还要煎熬，上网对答案，我发现，我的数学填空题一塌糊涂，政治也有好多选择题做错了，我好烦躁。每天担心，刷题也刷不进去。明明是要准备机试的，可是我还是止不住的在搜集调剂信息，中途我沉迷了王者荣耀，两天打到黄金一，之后慢慢上了白金，然后

。。。。。。

我开始教我的哥哥写代码。他学的还是很起劲，但是不知道现在如何，应该是放弃了吧。真不知道他的前路如何。又怕他想不开，会出事。想要的多，缺不想要付出，这该怎么办，没得办法。

成绩出来

真是让我好等啊！竟然考那么高，也算是值了吧，后来机试面试一帆风顺，几乎不可思议，找导师倒是有些曲折，但最终的结果是满意的。今后就要去做人工智能了，真是开心的事情。至少比写代码开心多了。

旅行

清迈之旅，应该是我这几年最开心的时光了，说不出又多么开心，无忧无虑的日子。多久没有这样过了。不想斗争，不想嫉妒与防备，在一个陌生的小城，好好度个假。还认识了两个新朋友，很有乐趣。

后记

希望研究生阶段好好学习，努力完成课业，尊重老师同学，向着自己的理想进发吧！
