Redis源码分析——C语言面向对象编程

发表于 2017-04-03 | 分类于 技术

github上找到的一个最早的版本，1.30，这个版本里面已经加入了对很多不同类型的支持，还有一个自己管理内存的 VM，不过总体说来 Redis 的代码还是非常简单易懂的，这个系列不准备一行行的分析代码，就把 Redis 实现的亮点说一下吧。今天分析第一个亮点，Redis 虽然是使用 C 写的，但是其内部的构造其实是面向对象的。

Redis 内部简单来说就是一个大哈希表，dict 就是装这个哈希表的地方。但是对于不同的类型的 哈希表哈希函数是不同的，key、value 的构造函数和析构函数肯定也是不一样的（Redis 支持 String、List、Set 的数据结构），但是从某个 key 找到对应值的这种哈希表的操作是相同的，如果每个类型都重新写一个哈希表的确是太烦了，这个问题使用 Java，C++ 等支持面向对象的语言很容易就解决了，比如写一个 dict 的抽象类，把 key，value的构造和析构函数以及哈希函数写成抽象接口，然后把查找这种函数利用抽象接口实现出来，然后再搞几个 StringDict子类、ListDict 子类来继承这个父类，问题就解决了。

考研学习的一些总结

发表于 2017-04-03 | 分类于 生活

考研终于过去了，过去的几个月大概比我过去的几年都努力，最后效果也不错，总结一下各种备考的经验吧，对以后的学习还是很有帮助的。

手机扔掉

只有不带任何电子设备，才能好好学习，手机平板和电脑统统扔掉。

带着手机的时候总是想打开来玩玩，打开刷会知乎微博什么的，半小时就没有了，而且中断了自己之前的学习状态，重新投入进去又要好一会，如果加上上厕所打水中午吃饭这种不可避免的中断，其实连续而不间断的学习时间就非常之少了。

其实平板对于我考研的帮助还是蛮大的，我把平板上的所有用不到的 App 全部卸载掉了，主要留下的就是 Oplayer 这个播放器，这个播放器最好用的功能有两个，一个是 1.5 倍速看视频，一个是支持 samba，可以直接从我的小米路由器里下载文件。第一个功能可以最快速的过完课程内容，第二个功能可以简化下载的时间。另外一个作用就是学习的时候有问题可以用平板来查一下，很多攻略都是我在平板上看的。不过到了中后期平板也扔掉了，这个东西有时候也会浪费很多的时间。

清迈的意外之旅

发表于 2017-04-01 | 分类于 生活

考研复试结束，找导师的时候，看到鸽子超又放了别人鸽子，在朋友圈里贩卖自己的清迈旅行，想想之后确实没什么事，于是接手了俞超的鸽子团。团里除了我还有三个人彤彤，嘉敏和向乾，我们几个人没有老司机带路，也没有做任何攻略，等到了旅行开始那天，就直接上了去清迈的飞机，开始了这场意外的旅行。

初到清迈

到清迈是当地九点多，时间还早，泰国走落地签的话还要 1000 泰铢的签证费，这个其实可以省下来的，国内办签证只要几块钱的样子。办理入境的人比较多，排队排了两个小时，终于入境了。

接我们的导游找个这辆车真是彪，一路上开的飞快，清迈机场和市区不算太远，没一会我们就看到了两旁的市集，很像电影里面八十年代的样子，清迈的城市设计是真的耿直，就是下面这个小方块，用河围起来，横竖都是一千米。

到了清迈新鲜感特别足，放下行李就到处逛，旁边的建筑让我想到了很多老电影，充满了怀旧的感觉，小时候我家那个小城大概就是这个样子。清迈城里到处都是7-11，7-11大概是这里最具现代感的商店了，而且里面还可以用支付宝付款，对于没带信用卡的我来说真是一个好消息。
